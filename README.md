# Birdwatching

## Prerequisites

- Install: [Packer](https://developer.hashicorp.com/packer/install), [Vagrant](https://developer.hashicorp.com/vagrant/install) and [Ansible](https://docs.ansible.com/projects/ansible/latest/installation_guide/installation_distros.html)
- Ensure that your system doesn't have any existing VMs with this names: source, database, web-server-1, web-server-2 or load-balancer

## Setup Guide

1. Add .env to `web-servers` folder
2. Execute `run-infrastructure.sh`
